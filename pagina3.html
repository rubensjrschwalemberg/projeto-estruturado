Comandos GIT com flow



Comando Utilidade
git flow init inicia o git flow
git branch verificar os ramos que foram criados
git flow init iniciar o git flow
git push origin master publicar no repositório master online
git push origin develop publica no repositório master o novo branch develop
git flow feature start exemplo_1 cria uma feature exemplo_1
git flow feature finish exemplo_1 finaliza a feature e vai para o ramo develop
git flow release start 0.12.0 inicia release 0.12.0
git flow release publish 0.12.0 publica o release no github online para que mais desenvolvedores possam trabalhar no arquivo
git pull origin release/0.12.0 atualiza o repositório local
git flow release finish 0.12.0 finaliza o release
git push origin develop publica o ramo develop
git push origin 0.12.0 publica a tag
git checkout master sair para o ramo master
git checkout master sair para o ramo master
git flow hotfix start 0.13.0 cria um ramo separado do master chamado hotfix
git flow hotfix publish 0.13.0 publicando alterações no repositório master
git flow hotfix finish 0.13.0 finaliza o hotfix
