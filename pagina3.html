Comandos GIT com flow



Comando Utilidade
git clone "ssh" nome da pasta clona o repositório do github online na sua máquina
cd "nome do projeto" abrir o projeto
git flow init iniciar o git flow
git add . adiciona os arquivos para o commit
git commit -am "nome do arquivo" salva as alterações no repositório com um nome específico
git push origin master salva as alterações do seu repositório para o repositório do github
